# Complement

## Problem

Given a DNA sequence, obtain the corresponding complementary DNA sequence.

## Overview

In DNA double helix structures, base pairing is specific: Adenine (A) always pairs with Thymine (T), and Cytosine (C) always pairs with Guanine (G). The complement function converts each base to its pairing partner.

---

## Method

The complement is generated by mapping each nucleotide to its partner:
- `A` / `a` $\leftrightarrow$ `T` / `t`
- `C` / `c` $\leftrightarrow$ `G` / `g`

---

## Complexity

Time Complexity:

O(n)

Space Complexity:

O(n)

where:

n = len(sequence)

---

## Example

Input:

```python
comp_seq = Complement("AAAACCCGGT")
```

Output:

```text
"TTTTGGGCCA"
```

---

## Implementation Notes

The implementation utilizes Python's built-in `str.translate` with a translation table generated by `str.maketrans` for fast, low-level translation of characters. It preserves case sensitivity.
